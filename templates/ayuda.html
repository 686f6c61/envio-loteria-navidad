Acerca de este Proyecto

Esta aplicación web es un proyecto educativo desarrollado con fines de aprendizaje y demostración. No está destinada para uso en producción o para manejar datos reales de lotería. Puede contener errores o imprecisiones y no debe ser utilizada para transacciones reales o manejo de información sensible.

Funcionamiento de la Aplicación

  1. Selección de Lotería: Elige el tipo de lotería para la participación.
  2. Datos del Remitente: Ingresa la información del organizador de la participación.
  3. Datos del Décimo: Introduce los detalles del décimo de lotería.
  4. Añadir Participantes: Agrega la información de cada participante y su porcentaje de participación.
  5. Envío de Correos: Al finalizar, la aplicación enviará correos electrónicos a cada participante con su información de participación.
  6. Resumen: Se genera un resumen para el organizador con todos los detalles de la participación.

Manejo de Datos

Esta aplicación no almacena permanentemente ningún dato ingresado por los usuarios. La información se utiliza únicamente para generar los PDFs y enviar los correos electrónicos. Una vez completado el proceso, los datos se eliminan de nuestros sistemas.

Los únicos datos que podrían quedar registrados son los logs del servidor SMTP (SMTP2GO) utilizados para el envío de correos, que están fuera de nuestro control directo.

Descargo de Responsabilidad

Este es un proyecto educativo y no debe ser utilizado para manejar participaciones reales de lotería o información financiera sensible. Los desarrolladores no se hacen responsables de cualquier uso indebido o consecuencias derivadas del uso de esta aplicación.
