Ayuda y acerca de

Acerca de este proyecto

Esta aplicación web es un proyecto educativo desarrollado con fines de aprendizaje y demostración. No está destinada para uso en producción o para manejar datos reales de juegos de azar. Puede contener errores o imprecisiones y no debe ser utilizada para transacciones reales o manejo de información sensible.

Funcionamiento de la aplicación

Modo individual

  1. Selección de juego: Elige el tipo de juego de azar para la participación.
  2. Datos del remitente: Ingresa la información del organizador de la participación.
  3. Datos del boleto: Introduce los detalles del boleto o décimo según el tipo de juego.
  4. Añadir participantes: Agrega la información de cada participante y su porcentaje de participación.
  5. Envío de correos: Al finalizar, la aplicación enviará correos electrónicos a cada participante con su información de participación.
  6. Resumen: Se genera un resumen para el organizador con todos los detalles de la participación.

Modo empresas

  1. Selección de juego: Elige el tipo de juego de azar para la participación.
  2. Datos de la empresa: Ingresa el nombre de la empresa y la persona responsable de la participación.
  3. Datos del boleto: Introduce los detalles del boleto o décimo según el tipo de juego.
  4. Porcentaje total: Establece el porcentaje total a repartir entre todos los participantes.
  5. Carga de participantes: Sube un archivo CSV con los datos de los participantes o pégalos directamente.
  6. Envío de correos: Al finalizar, la aplicación enviará correos electrónicos a todos los participantes con su información de participación.

Formato del archivo CSV

El archivo CSV para cargar participantes debe tener el siguiente formato:

Nombre, Email, Teléfono (opcional)

Ejemplo:

Juan Pérez, juan.perez@ejemplo.com, 612345678
María García, maria.garcia@ejemplo.com, 623456789

Puedes descargar un archivo CSV de ejemplo para ver el formato correcto.

Manejo de datos

Esta aplicación no almacena permanentemente ningún dato ingresado por los usuarios. La información se utiliza únicamente para generar los PDFs y enviar los correos electrónicos. Una vez completado el proceso, los datos se eliminan de nuestros sistemas.

Los datos de participaciones se eliminan automáticamente después de 24 horas. Los únicos datos que podrían quedar registrados son los logs del servidor de correo utilizados para el envío de emails, que están fuera de nuestro control directo.

Descargo de responsabilidad

Este es un proyecto educativo y no debe ser utilizado para manejar participaciones reales de juegos de azar o información financiera sensible. Los desarrolladores no se hacen responsables de cualquier uso indebido o consecuencias derivadas del uso de esta aplicación.
